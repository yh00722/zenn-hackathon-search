#  防災訓練プラットフォーム「TownReady」

デモ動画：  
<https://youtu.be/0jKiGXZBFQ8>

##  1\. 概要：AIと地域防災訓練

「TownReady」は、自治会、学校、企業などが長年抱えてきた防災訓練の企画・準備における非効率性と形骸化という根深い課題に対し、Google Cloudの最先端AI技術を活用した解決策を提供するプラットフォームです。  
住所を入力するだけで、地域固有のハザードマップと連動した訓練台本・役割表・避難導線・安全レビュー までを自動生成。これにより、全国4.5万の町内会と700以上の自治体が直面する「属人的な準備作業」と「形骸化した訓練」を、AIとクラウドの力で変革します。

**TownReadyが自動生成する「地域特化型訓練一式」:**

  * **訓練台本** （Markdown形式）：具体的な行動指示、役割分担、タイムラインを含む詳細なシナリオを生成。
  * **役割分担表** （CSV形式）：参加者の属性や訓練規模に応じた最適な役割分担。
  * **避難経路とポスター** （画像生成AI「Imagen」使用）：避難経路と配布用イラストを生成。
  * **安全レビュー** ：ガイドラインに基づく安全レビュー観点の提示

TownReadyは、単なる自動化ツールに留まらず、訓練の質そのものを向上させ、参加者の防災意識の向上を促すことで、地域社会全体のレジリエンス強化に貢献します。

##  2\. ターゲットと課題

「TownReady」は、防災訓練の企画・実施における、プロセス改善を目的とします。地域コミュニティ、学校、企業といった異なる組織において、顕在化している課題の根底には「準備の非効率性」「参加者の無関心」「効果測定と改善サイクルの欠如」という共通の構造が存在します。これらの課題は、全国で生じている課題でありながら、いまだに効果的な解決策が与えられていない、新規性のある課題といえます。

###  2.1. 自治会・地域ネットワーク：属人化と担い手不足の深刻化

自治会や町内会における防災訓練は、地域の安全を守る上で極めて重要ですが、その準備は特定の役員やボランティアに大きく依存しています。訓練シナリオの作成、役割分担、住民への広報、資機材の手配など、多岐にわたる作業が手作業で行われるため、担当者には膨大な時間と労力がかかります。この「準備の属人化」は、特に役員の高齢化や担い手不足が深刻化する現代において、訓練自体の継続を困難にする大きな要因となっています。

このような状況は、訓練の質を低下させるだけでなく、地域全体の防災力向上を阻害する「静かな危機」を生み出しています。また、多文化共生が進む地域では、外国人住民への情報伝達不足も課題となっており、従来の日本語回覧板だけでは訓練情報が行き渡らないケースが散見されます[1]。

###  2.2. 自治体の防災担当者：多様なニーズへの対応と効果測定の困難さ

自治体の防災担当者は、住民の多様化するニーズ（多言語対応、要配慮者への配慮など）に応えつつ、より実効性のある訓練を企画・推進する責任を負っています。しかし、限られたリソースの中で、各地域の特性に合わせた訓練をきめ細やかに支援し、その効果を定量的に測定することは容易ではありません。

また、訓練後の効果測定や改善提案が行われている事例は少なく、特に地域レベルでは、参加率や避難時間のデータ収集すら行われていないことが多いと指摘されています[2]。これにより、訓練が本当に有効であったのか、どのような改善が必要なのかが不明確なまま、次の訓練が実施されることになります。

###  2.3. 学校の教職員：訓練の形骸化とマンネリ化

学校では法令に基づき定期的な避難訓練が実施されますが、その準備には教職員に相当な工数がかかります。しかし、その内容は毎年同じような流れを繰り返すことが多く、訓練の形骸化やマンネリ化が指摘されています。

生徒側も「どうせ毎回同じ」と捉えがちで、主体的な参加や深い学びへと繋がりにくい現状があります。特に中高生以上では訓練への関心低下も見られ、形式上参加しても真剣に取り組まない学生もいると報告されています。

##  3\. TownReadyが提供するソリューションと特徴

「TownReady」は、前述の防災訓練が抱える根深い課題に対し、AIとデータ駆動型のアプローチを組み合わせることで、これまでにない具体的かつ効果的な解決策を提供します。単なる自動化に留まらず、訓練の質そのものを向上させ、持続可能な地域防災体制の構築に貢献します。

###  3.1. 企画工数の劇的削減：10分で「地域特化型訓練一式」を自動生成

TownReadyの最も画期的な点は、これまで数週間から数ヶ月を要していた訓練準備を、わずか数分で完了させる点にあります。ユーザーは、住所、参加者の属性（子供、高齢者、車椅子利用者）、想定する災害の種類といった基本情報を入力するだけです。すると、AIが以下の「地域特化型訓練一式」を自動で生成します。

  * **訓練台本（Markdown形式）** : 入力された情報に基づき、具体的な行動指示、役割分担、タイムラインを含む詳細な台本を生成します。これは、現場での混乱を最小限に抑え、スムーズな訓練実施を可能にします。
  * **役割分担表（CSV形式）** : 参加者の属性や訓練の規模に応じて、最適な役割分担を提案します。これにより、担当者の負担を軽減し、訓練運営の効率化を図ります。
  * **避難導線とポスター画像の生成（「Imagen」使用）** : 地図情報と避難経路を組み合わせて、避難経路ポスターとして、配布可能。

###  3.2. 参加率と学習効果の向上

訓練の形骸化を打破し、参加者の防災意識と学習効果を向上させるため、TownReadyは生成AIの力を最大限に活用します。

  * **視覚的なコンテンツ** : 画像生成AI「Imagen」によるポスターは、従来の文字情報中心の資料と比較して、参加者の関心を強く引きつけます。

###  3.4. 科学的根拠に基づく安全性と信頼性

TownReadyが生成する訓練内容は、国や自治体が定める防災ガイドライン、避難所運営マニュアル、地域のハザードマップといった信頼性の高い情報を「知識ベース」としてVertex AI Searchに格納し、これを参照してコンテンツを生成します。これにより、生成される訓練内容が常に最新かつ科学的根拠に基づいたものであることを保証し、安全性と信頼性を確保します。

  * **ガイドラインに基づく安全レビュー** : 訓練シナリオや避難経路の生成と共に、知識ベースに基づく安全レビューを自動実行。（チェックリスト照合による自動判定は拡充予定。）これにより、人為的なミスや知識不足による不適切な訓練内容の生成を防ぎます。

これらのアプローチにより、TownReadyは防災訓練の「質」と「効率」を同時に最大化し、地域社会全体のレジリエンス向上に貢献します。

##  4\. システムアーキテクチャ

「TownReady」は、堅牢性、拡張性を意識して作成したアプリケーションです。フロントエンド、バックエンド、そして非同期処理を行うワーカーが連携して動作し、大量の処理要求にも柔軟に対応できる設計となっています。

コンポーネント | Google Cloud サービス | 実装の特徴  
---|---|---  
フロントエンド | Cloud Run (Next.js) | 署名URL自動再発行、60秒クールダウン制御  
API層 | Cloud Run (FastAPI) | OIDC認証、冪等性保証、指数バックオフ実装  
Worker層 | Cloud Run + Pub/Sub | 非同期タスク連鎖、障害時の自動リトライ  
データ永続化 | Firestore Native | リアルタイム同期、トランザクション保証  
AI処理 | Vertex AI (Gemini/Imagen) | JSONモード活用、コスト上限制御  
知識検索 | Vertex AI Search | 安全基準との自動照合、根拠提示  
ストレージ | Cloud Storage | 署名URLによる安全な配布  
  
###  アーキテクチャのポイント：実装品質と拡張性

  * **サーバーレスアーキテクチャ** : Google Cloud Run を全面的に採用することで、インフラ管理のオーバーヘッドを最小限に抑えつつ、アクセス数や処理量に応じて自動的にスケールする高い柔軟性を実現しています。これにより、運用コストを最適化し、プロンプト改善などアプリケーションロジックに集中できます。
  * **非同期処理による高効率** : 時間のかかるAIによるコンテンツ生成処理は、Google Cloud Pub/Sub を介してバックグラウンドのワーカー（Cloud Run）で非同期に実行されます。これにより、ユーザーは生成完了を待つことなく次の操作に進むことができ、優れたユーザーエクスペリエンスを提供します。また、システム全体の応答性を高め、リソースの効率的な利用を可能にしています。
  * **マルチモーダAIの活用** : Google Cloudの最先端AIモデル（テキスト生成のGemini、画像生成のImagen）をシームレスに統合し、多様な形式の訓練コンテンツを生成します。これにより、従来の防災訓練では実現困難だった、視覚的に訴えかけるリッチなコンテンツ提供を可能にし、訓練の質を向上させます。
  * **堅牢なデータ管理とスケーラビリティ** : 訓練データやメタデータはFirestoreに、生成されたファイル（台本、ポスター、ビデオなど）はCloud Storageに保存されます。これらのサービスは、高い可用性と耐久性を持ち、将来的なデータ量の増加や機能拡張にも柔軟に対応できるスケーラビリティを保証します。
  * **知識ベースによる信頼性** : Vertex AI Search を活用した知識ベースには、国や自治体の防災ガイドライン、ハザードマップなどの信頼性の高い情報を格納。AIはこれを参照してコンテンツを生成するため、常に最新かつ科学的根拠に基づいた訓練内容を提供し、その信頼性と安全性を担保します。

これらのアーキテクチャ設計により、TownReadyは高い実装品質と将来にわたる拡張性を確保し、持続可能な地域防災プラットフォームとしての役割を果たすことができます。

##  5\. 市場インパクトと拡張性

###  5.1 対象市場の規模

  * 全国4.5万超の町内会・自治会
  * 700以上の自治体防災担当部署
  * 学校（小中高・大学）: 法令により定期的な訓練が義務付け
  * 企業・オフィス: 消防法に基づき年1回以上の防火避難訓練が義務付け

###  5.2 将来的な拡張構想

  * BigQuery連携: 訓練データの蓄積・分析による改善提案の自動化
  * 全国展開: RegionContextの自動同期によるカバレッジ拡大
  * 多言語展開: 在日外国人コミュニティへの横展開
  * BCP連携: 企業の事業継続計画と連動した総合防災プラットフォーム化

##  6\. 最後に：地域防災の未来を拓く「TownReady」

「TownReady」は、防災訓練が抱える課題に対し、AIとクラウド技術を駆使したアプローチで挑みます。近年の豪雨、地震などの大規模な災害に備え、常日頃からの準備の重要度が高まっています。  
AIツールを活用したシステムが、避難訓練の質を高め、災害時に役立つ知識と経験をもたらすことで、誰もが安心して暮らせるようサポートできれば、と思います。  
「TownReady」による、訓練準備の効率化、訓練の質の向上、そしてデータに基づいた継続的な改善サイクルを通じて、地域社会全体の防災力を底上げに貢献します。
