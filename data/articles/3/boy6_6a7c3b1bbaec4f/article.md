はじめに

高齢化社会が進む中、高齢者の方々の孤独感や健康管理の課題が深刻化しています。特に一人暮らしの高齢者は、家族とのコミュニケーション不足や健康状態の変化に気づきにくい状況にあります。

本プロジェクト「AI Voice Journal」は、高齢者の方でも簡単に使える音声日記アプリを通じて、これらの社会課題を解決することを目指しています。

プロジェクト概要

対象ユーザー像

プライマリターゲット

  * 65 歳以上の高齢者（約 3,600 万人）
  * 特に一人暮らしの高齢者（約 670 万人）
  * 手が不自由で文字入力が困難な方
  * 記憶力の衰えで日記を忘れがちな方

セカンダリターゲット

  * 高齢者の家族（約 2,000 万人）
  * 介護施設・デイサービス
  * 医療機関・介護事業者

解決する課題

  1. コミュニケーション不足

     * 家族との会話機会の減少
     * 離れて暮らす家族への心配
  2. 健康管理の困難

     * 体調変化の早期発見が困難
     * 感情の変化に気づきにくい
  3. デジタルデバイド

     * 複雑な操作が苦手
     * 文字入力が困難
  4. 孤独感の増大

     * 社会との接点の減少
     * 認知機能の維持が困難

ソリューションと特徴

AI Voice Journalは、以下の 4 つのコア機能で課題を解決します：

  1. 音声入力機能

  * 話すだけで日記記録: ボタン一つで簡単録音
  * 自動文字起こし: Web Speech API による高精度変換
  * 手が不自由でも操作可能: 大きなボタンとシンプルな画面

  2. AI 分析機能

  * 感情状態の分析: Google Gemini AI による感情スコア算出
  * 健康状態のチェック: 体調変化の早期発見
  * 変化の可視化: グラフで感情・健康の推移を表示

  3. 家族共有機能

  * リアルタイム共有: 家族と日記を安全に共有
  * プッシュ通知: 家族に高齢者の状況を即座に通知
  * 安心の提供: 離れていても心配りが可能

  4. 簡単操作設計

  * 高齢者に優しい UI: 大きなボタンとシンプルな画面
  * 直感的な操作: 迷わず使える設計
  * PWA 対応: アプリのように使える

システムアーキテクチャ

アーキテクチャの特徴

フロントエンド

  * React + TypeScript による型安全な開発
  * PWA 対応でオフライン利用可能
  * 高齢者に優しいレスポンシブデザイン

バックエンド

  * Supabase による認証・データベース・ストレージ
  * Edge Functions によるサーバーレス API
  * リアルタイム同期で家族間の即座な情報共有

AI・分析

  * Google Gemini AI による感情・健康状態分析
  * Web Speech API による音声認識・文字起こし
  * 自動分析による日記内容の感情スコア算出

通知・決済

  * Firebase FCM によるプッシュ通知
  * Stripe による安全な決済処理
  * 自動課金によるサブスクリプション管理

技術スタック

フロントエンド

  * React 18 + TypeScript: モダンな UI 開発
  * Tailwind CSS: 効率的なスタイリング

バックエンド

  * Supabase: 認証・データベース・ストレージ

AI・分析

  * Google Gemini AI: 感情・健康状態分析

通知・決済

  * Firebase Cloud Messaging: プッシュ通知
  * Stripe: 決済処理・サブスクリプション管理
  * VAPID: Web プッシュ通知認証

デプロイ・ホスティング

  * Vercel: フロントエンドホスティング
